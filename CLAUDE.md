# Plane AI Testing Project 协作规则

## 团队分工

1. ChatGPT 负责总控、任务拆解、结果分析和质量验收。
2. Codex 负责关键代码、项目结构、重要文档和质量复查。
3. Claude Code 负责 Linux 执行、Docker 部署、日志采集和测试运行。

## 真实性与安全

1. 未经实际执行，不允许填写“测试通过”。
2. 不允许提交密码、Cookie、API Key、Personal Access Token 或真实环境变量。
3. 执行失败时必须保留原始错误，不得用虚假成功结果替代。
4. 所有证据必须能够对应到具体命令或测试场景。

## 修改与协作

1. 修改重要文件前先读取现有内容。
2. 每次只处理当前阶段任务，未经确认不得扩展到后续阶段。
3. Codex 与 Claude Code 不得同时修改同一个文件。
4. 不得删除、覆盖或弱化既有断言、测试资产或证据来制造成功结果。
5. 新增依赖、框架或配置后，先完成相应的静态或聚焦校验，再纳入后续测试流程。

## Day 1 边界

本阶段只初始化项目结构、配置、文档模板与安全检查脚本。不得伪造测试数据、部署状态、容器状态、页面截图、接口响应或测试结论；不得执行 Plane 部署或提交 Git。
